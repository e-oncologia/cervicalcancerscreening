DEEP LEARNING / INTELIGENCIA ARTIFICIAL

4.2 Desarrollo de un algoritmo de aprendizaje automático

El desarrollo de un algoritmo de aprendizaje automático es un proceso experimental iterativo mediante el cual los modelos (o algoritmos) deben ser entrenados (se le enseña al modelo a identificar el resultado de interés en función del estándar de referencia utilizado, por ejemplo, un diagnóstico histológico) y ajustados en función del contexto y los datos en cuestión. Una vez entrenado, el algoritmo debe ser probado con datos ciegos para garantizar que los datos originales no estaban sesgados. Se debe tener cuidado para evitar modelos que sean demasiado específicos para un conjunto de datos dado (sobreajuste), ya que este hecho afectaría la generalizabilidad del modelo [85]K.
