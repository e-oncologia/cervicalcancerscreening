Introducción

Hoy en día existen varias pruebas disponibles dirigidas a las diferentes etapas del proceso de carcinogénesis de cuello uterino. Este módulo centra la discusión en aquellas pruebas adecuadas para el cribado primario (es decir, la prueba inicial utilizada para cribar a la población asintomática).

Tal y como está descrito en el módulo 2, el cribado primario tiene como objetivo maximizar la detección de lesiones precancerosas de cuello uterino que podrían progresar a cáncer si no se tratasen. Idealmente, las pruebas deberían tener una alta sensibilidad (identificar si la enfermedad en cuestión está presente con un resultado positivo de la prueba) y una alta especificidad (no clasificar erróneamente a pacientes sanas sin la enfermedad con un resultado positivo de la prueba). Estos dos requisitos (sensibilidad y especificidad) deben equilibrarse para maximizar la efectividad y minimizar el coste y el daño asociado al tratamiento y pruebas complementarias. Para una prueba de cribado de primera línea, se prefiere una alta sensibilidad a una alta especificidad para minimizar el número de falsos negativos.

Sin embargo, para reducir la derivación innecesaria a pruebas adicionales para el diagnóstico final y/o tratamiento (falsos positivos) en poblaciones cribadas mediante prueba primaria con especificidad limitada, las personas que dan positivo vuelven a ser examinadas mediante una prueba más específica. Este paso se llama TRIAJE y puede estar integrado dentro de una prueba primaria en sí.

Ejemplo

Se podría utilizar una prueba del VPH que proporciona resultados agrupados solamente para la positividad a los tipos del VPH que detecta la prueba y utilizar la citología como prueba de triaje (dos pruebas separadas). Por otra parte, algunos ensayos o pruebas aportan información específica sobre el genotipo de determinados tipos de VPH con mayor riesgo, como el VPH 16 y el VPH 18, mientras que el resto de los tipos se agrupan juntos (otros VPH de alto riesgo). Esta distinción permite un manejo clínico diferenciado.

Este módulo se centra en el cribado primario del cáncer de cuello uterino mediante citología y/o pruebas de VPH.
